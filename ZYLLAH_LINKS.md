## Instrução de uso

Quando precisar ler um documento:
1. Localizar o link neste índice
2. Substituir `[REPO-PUBLIC]` pela URL base do repositório público após hospedar
3. Usar web_fetch para buscar o conteúdo

Quando precisar de documento privado:
- Colar o conteúdo diretamente na conversa
- Nunca hospedar publicamente
