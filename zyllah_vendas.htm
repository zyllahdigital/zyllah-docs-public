(function() {

  var SENHA = 'V#3x7KjnLDRf';

  var KEY = 'zyl_zyllahvendas';

  if (sessionStorage.getItem(KEY) === '1') return;

  var tela = document.getElementById('senha-tela');

  if (tela) tela.style.display = 'flex';

  document.body.style.overflow = 'hidden';

  window.verificarSenha = function() {

    var inp = document.getElementById('senha-input');

    var err = document.getElementById('senha-erro');

    if (inp.value === SENHA) {

      sessionStorage.setItem(KEY, '1');

      tela.style.transition = 'opacity 0.3s';

      tela.style.opacity = '0';

      setTimeout(function() { tela.style.display = 'none'; document.body.style.overflow = ''; }, 300);

    } else {

      err.textContent = 'Senha incorreta.';

      inp.value = ''; inp.focus();

      setTimeout(function() { err.textContent = ''; }, 2000);

    }

  };

  document.addEventListener('DOMContentLoaded', function() {

    var inp = document.getElementById('senha-input');

    if (inp) { inp.focus(); inp.addEventListener('keydown', function(e) { if (e.key==='Enter') verificarSenha(); }); }

  });

})();

<section class="secao" id="objecoes">

  <div class="wrap">

    <span class="secao-tag">Seção 04 · Respostas prontas</span>

    <h2 class="secao-titulo">As objeções<br><em>mais comuns</em></h2>

    <p class="secao-sub">Toda objeção é um pedido de mais informação. Não é rejeição — é a pessoa tentando entender se vale a pena. Responda com calma, sem defensividade.</p>

 

    <div class="objecoes-grid">

      <div class="objecao">

        <div class="objecao-q">"Está caro. Outra agência cobra menos."</div>

        <div class="objecao-r">Faz sentido comparar. A diferença é o que você obtém: a maioria das agências entrega volume — posts sem identidade, sem estratégia. Nós entregamos um sistema. A pergunta não é o custo da Zyllah, mas o custo de uma presença que diminui o que você representa. <strong>Um paciente do perfil errado que abandona o tratamento custa mais que seis meses de honorários.</strong></div>

      </div>

      <div class="objecao">

        <div class="objecao-q">"Preciso pensar. Vou avaliar mais."</div>

        <div class="objecao-r">Claro. Só peço que avalie uma coisa específica: <strong>o custo de adiar.</strong> Cada semana com uma presença que não reflete o que você é, é uma semana com pacientes chegando pelo motivo errado — ou não chegando. Não tenho pressa, mas o seu tempo tem valor.</div>

      </div>

      <div class="objecao">

        <div class="objecao-q">"Não sei se o digital vai trazer pacientes de verdade."</div>

        <div class="objecao-r">É uma dúvida legítima. Por isso nossa meta inicial não é volume — é qualidade. <strong>O primeiro objetivo é que a sua presença digital atraia o paciente certo, não qualquer paciente.</strong> Aumento de volume vem depois, quando a base está construída. E isso é mensurável — você saberá exatamente o que mudou.</div>

      </div>

      <div class="objecao">

        <div class="objecao-q">"Já tentei antes e não funcionou."</div>

        <div class="objecao-r">Essa é a objeção mais importante. Me conta o que aconteceu? Na maioria dos casos, o problema foi execução sem estratégia — conteúdo sem identidade, posts sem posicionamento, ou uma agência que não entendeu o nicho de saúde. <strong>A Zyllah foi construída especificamente para esse problema.</strong></div>

      </div>

      <div class="objecao">

        <div class="objecao-q">"Não tenho tempo para gravar vídeo toda semana."</div>

        <div class="objecao-r">Esse é exatamente o sistema que a Zyllah resolve. <strong>Fazemos sessões concentradas uma vez por mês</strong> — 2 horas que geram conteúdo suficiente para o ciclo inteiro. Você não gerencia as redes, não pensa no próximo post, não grava nada fora do agendado.</div>

      </div>

      <div class="objecao">

        <div class="objecao-q">"Sou novo, ainda não tenho muitos pacientes."</div>

        <div class="objecao-r">Melhor momento para começar. <strong>Construir a identidade certa desde o início é muito mais eficiente do que tentar corrigir uma reputação digital já consolidada errada.</strong> Os primeiros 100 pacientes formam a base da sua cartela para sempre — vale investir em quem são eles.</div>
