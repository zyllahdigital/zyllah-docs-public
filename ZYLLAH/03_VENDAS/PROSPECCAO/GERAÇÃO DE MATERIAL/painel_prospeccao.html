Inspiração editorial
O que é
O mesmo vídeo que serve para prospectar serve para calibrar o editorial da Zyllah. O pipeline lê os posts vistos e identifica: o que está em alta, o que está faltando, o que a Zyllah deveria abordar esta semana.
Output do pipeline — editorial
O arquivo AAAMMDD_editorial.md em inbox/saidas/ contém:
✓
Tendências identificadas no nicho
✓
Oportunidades editoriais com ângulo Zyllah
✓
O que NÃO fazer (saturado)
✓
Pauta da semana: 3 temas prontos

Integração com a pauta
Após gerar o editorial, abrir o hub → Pauta → usar as sugestões para criar os itens da semana. O pipeline sugere; você decide.
